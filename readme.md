# Loan Status Prediction System

## Project Overview
The Loan Status Prediction System is a Machine Learning project that predicts whether a loan application will be approved or rejected based on customer financial and demographic details. The project demonstrates the complete machine learning workflow from data preprocessing to model deployment.

---

## Problem Statement
Banks and financial institutions receive many loan applications daily. Manual verification takes time and may lead to inconsistent decisions. This project helps automate the loan approval prediction process using machine learning algorithms.

---

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Hyperparameter Tuning
- Model Evaluation
- Streamlit Web Application
- Real-Time Loan Prediction
